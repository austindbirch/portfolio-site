# Easter Eggs

If you're reading this md, congrats! You'll probably understand these jokes.
The terminal has a few surprises for devs who go exploring.

Here are some to get you started:

- `sl` - a classic
- `hack` - initiate hack sequence
- `brew` - need some coffee?
- `man` - very recursive
- `python` - in this RAM economy?
- `uname` - ...2038?

There are plenty more. Happy hunting.
